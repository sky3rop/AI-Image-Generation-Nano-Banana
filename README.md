AI Image Studio (Nano Banana)

This is a simple, mobile-friendly web application that uses the Gemini "Nano Banana" (gemini-2.5-flash-image-preview) model to generate and edit images. It also uses the gemini-2.5-flash-preview-09-2025 model to suggest creative prompts.

This entire application is self-contained in a single nano-banana-studio.html file.

Features

Generate Mode: Create new images from a text prompt.

Edit Mode: Upload your own photo and use a text prompt to describe your edits (e.g., "make my hair blue," "add a hat," "change the background").

✨ Get Idea: A feature that uses a text-based Gemini model to suggest creative prompts for either generating or editing an image.

Mobile-Friendly: The UI is built with Tailwind CSS and is fully responsive for use on both desktop and mobile devices.

Self-Contained: All HTML, CSS, and JavaScript are in a single file, making it easy to deploy or use.

How to Use

Open the nano-banana-studio.html file in any modern web browser.

Toggle between Generate and Edit modes.

Type your prompt and click the action button.

(Optional) Click "✨ Get Idea" if you're looking for inspiration.

Technologies Used

HTML5: For the page structure.

Tailwind CSS: For all styling and responsive design.

JavaScript (ES6+): For all application logic and API calls.

Google Gemini API:

gemini-2.5-flash-image-preview (Nano Banana): For image generation and editing.

gemini-2.5-flash-preview-09-2025: For text-based prompt suggestions.

This app was built as a single-file web application.
